# chatter-box-fe

## Getting Started

- Clone the repo
  ```
  git clone https://github.com/supriya-kotturu/chatter-box-fe.git
  ```
- Install the dependencies
  ```
  npm install
  ```
- Build the files with vite
  ```
  npm run build
  ```
- Locally preview the production build
  ```
  npm run preview
  ```
- Run the application in development mode
  ```
  npm start
  ```
